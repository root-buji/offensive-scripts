--- Apache Solr (​https://lucene.apache.org/solr/​)



- Envío un **dataConfig malicioso** a Solr
    
- Solr acepta la configuración
    
- Solr descarga un XML remoto
    
- El XML activa transformer
    
- El script se ejecuta en el servidor
    
- Obtengo **RCE**
